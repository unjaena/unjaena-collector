APK Downgrade Forensic Acquisition - Downgrade APK Repository
=============================================================

Place older APK versions here for forensic APK downgrade acquisition.
The APK must be an older version where allowBackup=true in AndroidManifest.xml.

Naming convention:
  <package_name>.apk
  Example: com.kakao.talk.apk

Or use a directory:
  <package_name>/some_version.apk
  Example: com.kakao.talk/com.kakao.talk_9.0.0.apk

Supported packages:
  - com.kakao.talk          (KakaoTalk)
  - com.whatsapp            (WhatsApp)
  - org.telegram.messenger  (Telegram)
  - jp.naver.line.android   (LINE)
  - com.discord             (Discord)
  - org.thoughtcrime.securesms (Signal)
  - com.nhn.android.band    (BAND)
  - com.facebook.orca       (Facebook Messenger)
  - com.tencent.mm          (WeChat)

Where to download old APKs:
  - APKMirror: https://www.apkmirror.com/
  - APKPure: https://apkpure.com/
  - Avilla Forensics: Built-in downgrade APK repository

IMPORTANT:
  - Verify APK signature matches the original developer
  - Use the oldest available version that still installs on the target device
  - The downgrade preserves app data (messages, contacts, etc.)
  - Original APK is automatically restored after backup extraction
