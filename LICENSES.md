- Windows builds may include pywin32, WMI, pytsk3, libewf-python, Android platform-tools, and native USB libraries.

- Linux builds use system libraries installed by the release workflow for Qt, OpenGL, and USB access.

## Server Boundary

This repository is a standalone acquisition client. The unJaena service communicates with the collector through authenticated HTTP/WebSocket APIs. Server-side parsing, AI analysis, retention, deletion, billing, and case-management systems are separate software components and are not included in this repository.
